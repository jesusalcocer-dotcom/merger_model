# Deal Structurer — Product Spec

**Status:** Draft
**Last updated:** 2026-03-20

## Overview

Tool for structuring M&A transactions. Parses natural language deal descriptions into structured data across four blocks, allows user editing, then generates a visual deal diagram with auto-detected regulatory and structural flags.

## Spec sections (to be populated)

1. Data schemas (entities, relationships, structure, consideration flows)
2. UI/UX flow (input → parse → edit → generate)
3. API integration (Claude parsing prompt, response schema)
4. Diagram rendering (node types, edge types, layout algorithm, color coding)
5. Auto-detection rules (CFIUS, minority issues, thin credit, carve-out/TSA)
6. Export (JSON, Excel integration with Summit model)
